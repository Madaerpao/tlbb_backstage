# TLBB 账号及 GM 管理运维后台 (重构高安全增强版)

这是一个极轻量级的 Web 账号与服务器管理工具，专门为天龙八部单机版和联机版设计。本项目在原版的基础上进行了深度重构与安全升级，将“玩家自助服务”与“GM运维后台”彻底分离，不仅提供了炫酷现代的 UI 界面，还达到了企业级的安全防护标准。

---

## 🌟 核心特色与优势

1. **玩家与管理员完全隔离的权限架构**：
   - **玩家自助端**：提供纯净的玩家自助服务入口，包括注册、改密、自救、解封、潜能分配等，没有任何 GM 工具的暴露。
   - **独立的 GM 运维控制台**：运维人员通过专属的 `/admin` 路由登录。通过严密的 **Session 签名鉴权**机制保护，非授权访问将被直接拦截。
2. **全新高端 UI 设计**：采用现代化的 Glassmorphism（玻璃拟物化）设计风格和深色护眼模式，支持微动画交互，告别传统后台的“土味”界面，且前后端统一风格。
3. **极简操作流程**：登录 GM 控制台后，无需在每次操作时重复输入密码，系统自动维护登录状态，极大提高运维效率。
4. **企业级安全防护**：
   - 彻底修复原版高危 **SQL 注入**漏洞，全部采用参数化查询。
   - 内置 **Flask Session** 服务端级别图形验证码，拦截机器恶意注册和撞库攻击。
5. **无需依赖特定登录器**：这是一个纯粹的 Web 应用程序，与游戏服务器完全解耦。

---

## 🎮 面向玩家的自助功能

首页入口供玩家自由访问，提供以下便捷服务：
- **账号自助服务**：在线注册新账号、安全修改登录密码。
- **地图卡死自救**：角色卡死在某个地图无法登录时，可以一键强制将角色传送回主城。
- **自助解封**：账号因异常被封停时尝试自助解除锁定。
- **在线潜能分配**：离线状态下在网页上直接给角色增加未分配的潜能点（默认修改 `t_char` 表的 `remainpoints` 字段）。

---

## 🛠 面向服主的 GM 运维控制台

服务器管理员专属后台，强大的远程运维能力：
- **入口路径**：访问 `http://您的IP:8881/admin` 进入登录页。
- **GM 专属充值**：一键为指定玩家发放游戏点数/充值点。
- **高级 GM 工具 (事件下发机制)**：
  - 发送全服公告
  - 发放任意物品给指定角色
  - 随意修改角色等级
  - 发放元宝给角色
  *(以上功能均基于无缝的 HTTP 轮询下发，游戏内无需重启，且玩家无需下线即可生效)*

## 🚀 部署与使用说明

### 1. 获取安装包

```shell
cd ~ && git clone https://gitee.com/zhao-kai135/tlbb_backstage.git .
```

### 2. 配置说明
打开 `flaskProject/myconf.py`，按照您的服务器环境进行修改：

```python
config = {
    'IP': '您的数据库Ip',             # 数据库所在IP，一般是Linux的IP地址
    'DB_PASSWORD': '您的数据库密码',    # MySQL 数据库密码
    'DB_PORT': 3306,                 # 数据库端口
    
    # 功能开关
    'register': True,                # 网页自助注册功能是否开启
    'modify_passwd': True,           # 修改密码功能是否开启
    'gm_point': True,                # GM在线发放点数功能是否开启
    
    # 管理员与接口配置（重要）
    'gm_tool_pwd': 'YourAdminPassword',  # 登录 /admin 的管理员密码，请务必修改为强密码！
    'private_key': 'YourPrivateKeyHere', # API 通信秘钥，提供给游戏内 Lua 脚本进行身份验证
    
    'my_website': 'https://www.zkwd888.ltd' # 您的主页地址，可用于页面底部的跳转
}
```

### 3. 运行项目
将修改后的代码上传至 Linux 服务器后，执行以下命令：
```shell
cd /root/tlbb_backstage/flaskProject
chmod -R 777 .
sh run.sh
```
> 此时您可以访问 `http://您的IP:8881` 查看玩家首页，或访问 `http://您的IP:8881/admin` 登录 GM 控制台。

### 4. 高级 GM 工具（服务端 Lua 心跳脚本）接入方法
为了让游戏服务端能自动轮询并执行发物品、发公告、发元宝等事件指令，请按照以下步骤配置心跳脚本：

1. **导入 Lua 脚本**：我们已将所需的 Lua 模板文件放至本项目的 `lua_scripts` 目录下。请将其内部的文件结构（`Public` 和 `Server`）合并至您游戏服务端的 `tlbb/` 目录中。
2. **场景心跳触发绑定**：在游戏中某个常驻场景（如大理、洛阳）添加一个 NPC，将其挂载脚本 `Server/SecondsTimer/XYNS_SecondsTimer.lua`（秒级心跳脚本），并使其每 2 秒触发一次请求。
3. **配置请求接口与通讯秘钥**：
   - 打开 `tlbb/Server/SecondsTimer/XYNS_SecondsTimer.lua` 脚本。
   - 将内部发起的网页请求地址更改为您刚部署的 Web 接口（如：`http://127.0.0.1:8881/api/get_event?privateKey=`）。
   - 将验证 KEY 替换为与您在 `flaskProject/myconf.py` 中 `private_key` 配置的完全一致。
4. **开启全局功能开关**：打开服务端的 `tlbb/Public/ScriptGlobal.lua`，确保将 `GMDATA_ISOPEN_GMTOOLS` 变量配置为 `1` (1=开启, 0=关闭)。
